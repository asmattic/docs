SSHFS¶

  • SSHFS
  • SSHFS Installation

To copy files over SSH or SFTP you can mount a remote directory using SSHFS.

Note

The Stack Exchange answer that let me to this has some other useful quick ways to forward a connection or copy over small files.

mkdir ~/cdn
sshfs cdn.example.com: ~/cdn
rsync -au ~/mystuff/dir/ ~/cdn/dir/
